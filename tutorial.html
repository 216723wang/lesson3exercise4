Basic P5.js setup

This project sets up a "hello world" (ok draw a circle) program using p5.js. It includes all the files you will need to produce a p5.js sketch without needing to add anything extra. It even include the addons for p5.js

Step-by-Step Instructions

  1. Type your sketch into the file called sketch.js
  2. If you don't need p5.dom.js or p5.sound.js you can delete them using the file explorer on the left
  3. If you do plan to use them, move each of the pieces you want to use below line 8 in the file index.html (this is the end comment line
  4. If you want to rename the sketch.js file, please remember to change the script tag in index.html
  5. NOTE: for outside media (graphics, sounds, video) resource, host your media and provide a url instead of uploading to your thimble project
